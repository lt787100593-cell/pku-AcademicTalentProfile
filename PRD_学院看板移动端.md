## 一、产品概述

### 1.1 产品背景

北京大学二级学院院长需要及时掌握本学院人事人才全貌，包括教职工结构、人才分布、人员流动等关键数据，以支撑日常管理和战略决策。目前缺乏便捷的移动端数据查看工具，院长需要通过PC端或人工汇报获取信息，效率较低。

### 1.2 产品定位

面向二级学院院长的**移动端人事数据看板**，提供实时、直观、可穿透的人事数据分析，帮助院长快速掌握学院人才全貌，支持管理决策。

### 1.3 核心价值

| 价值点 | 说明 |
|--------|------|
| 随时随地 | 手机端随时查看，不依赖PC |
| 一目了然 | 核心KPI一眼掌握 |
| 数据穿透 | 从概览到明细逐层深入 |
| 横向对比 | 了解本学院在全校的相对位置 |

### 1.4 技术方案

- **形态**：移动端H5页面
- **适配**：iOS/Android手机浏览器
- **访问方式**：通过北大企业微信或链接访问
- **数据更新**：每日自动同步人事数据
- **开发工具**：基于BI工具开发，专注数据分析

## 三、功能需求

### 3.1 功能架构

```
学院看板移动端（三Tab架构）
├── Tab1: 总览（30秒掌握全貌）
│   ├── 核心KPI卡片（含同比环比）
│   ├── 人员结构概览
│   ├── 人才概况摘要
│   └── 获奖亮点摘要
├── Tab2: 分析（多维度深挖）
│   ├── 年龄分布
│   ├── 性别分布
│   ├── 学历结构
│   ├── 职称结构
│   ├── 学缘结构
│   ├── 政治面貌
│   ├── 人才分布
│   └── 获奖分析
└── Tab3: 名单（人员明细查询）
    ├── 全部人员名单
    ├── 人才名单
    └── 获奖名单
```

### 3.2 功能模块详细说明

#### Tab1: 总览页面

**设计原则**：高信息密度、一屏可见、精致感

**布局结构**：

| 区域 | 内容 | 说明 |
|------|------|------|
| 筛选器 | 人员类别筛选 | 全部/专任教师/职员/合同制/博士后 |
| KPI卡片区 | 4个核心指标 | 在职人数、专任教师、高层次人才、本年获奖 |
| 人才概况 | 人才分布摘要 | 院士/长江/杰青/青年人才人数 + 查看完整名单 |
| 人员结构 | 环形图 | 各类人员占比 |
| 获奖亮点 | 获奖级别分布 | 国家级/省部级/校级 + 查看完整名单 |

**KPI卡片设计规范**：

| 指标 | 展示内容 | 说明 |
|------|---------|------|
| 在职人数 | 456人 | 含同比变化 ↑2.3% |
| 专任教师 | 234人 | 含同比变化 ↑1.8% |
| 高层次人才 | 45人 | 含同比变化 ↑3人 |
| 本年获奖 | 28人次 | 含同比变化 ↑5人次 |

**KPI卡片视觉规范**：
- 主数字：24px，Bold，北大红
- 标签：13px，Regular，灰色
- 变化值：12px，带颜色（绿色正向/红色负向）
- 卡片背景：白色，圆角16px，轻微阴影

---

#### Tab2: 分析页面

**设计原则**：图表为主、交互丰富、支持钻取

**8大分析维度**：

| 维度 | 图表类型 | 核心指标 | 全校对比 |
|------|---------|---------|---------|
| 年龄分布 | 柱状图 | 平均年龄、各年龄段人数 | 显示排名 |
| 性别分布 | 环形图 | 男女比例、人数占比 | 显示对比 |
| 学历结构 | 环形图 | 博士/硕士/本科占比 | 显示排名 |
| 职称结构 | 柱状图 | 正高/副高/中级/初级 | 显示排名 |
| 学缘结构 | 树图 | 本校/国内/海外 + TOP10院校 | 显示排名 |
| 政治面貌 | 环形图 | 党员/民主党派/群众 | 显示对比 |
| 人才分布 | 柱状图 | 院士/长江/杰青/青年人才 | 显示排名 |
| 获奖分析 | 柱状图+表格 | 按级别/年度分布 | 显示排名 |

**交互设计**：
- 顶部Tab切换8个维度
- 点击图表数据点可下钻到人员名单
- 每个图表旁显示全校排名徽章
- 支持按人员类别筛选

---

#### Tab3: 名单页面

**设计原则**：简洁、专注数据、可搜索

**筛选类型**：

| 类型 | 说明 |
|------|------|
| 全部 | 所有人员 |
| 人员 | 普通教职工 |
| 人才 | 高层次人才 |
| 获奖 | 获奖人员 |

**人员卡片展示字段**：

| 字段 | 说明 |
|------|------|
| 姓名 | 人员姓名 |
| 职称/职位 | 当前职称或职位 |
| 最高学历 | 博士/硕士/本科等 |
| 所属部门 | 院系/科室 |
| 来校时间 | 入职年份 |
| 人才标签 | 如"长江学者"、"杰青"等（如适用） |

**交互说明**：
- 支持按姓名、职称、学历搜索
- 支持按类型筛选（人员/人才/获奖）
- 点击单条可展开查看详情
- 不包含拨打电话等功能（专注数据分析）

---

#### 模块：人员进出（调整后）

**功能描述**：展示学院近三年人员流动情况（非30天）。

**展示内容**：

| 指标 | 说明 |
|------|------|
| 近三年入职总数 | 新入职人员数量 |
| 近三年减离总数 | 离职、调出人员数量 |
| 近三年净增减 | 入职-减离 |
| 年度趋势 | 折线图展示近三年每年进出 |
| 全校排名 | 本学院净增减在全校的排名 |

**交互说明**：
- 支持按人员类别筛选
- 点击可下钻到对应人员名单

---

#### 模块：人才情况（详细）

**功能描述**：展示学院高层次人才信息。

**人才分类**：

| 类别 | 说明 |
|------|------|
| 院士 | 中国科学院/工程院院士 |
| 长江学者 | 长江学者特聘教授/讲座教授 |
| 杰青 | 国家杰出青年基金获得者 |
| 青年人才 | 青年千人、青年长江等 |

**展示内容**：

| 指标 | 说明 |
|------|------|
| 人才总数 | 各类人才合计 |
| 各类别人数 | 院士/长江/杰青/青年人才 |
| 近3年引进趋势 | 折线图 |
| 人才名单 | 完整人才列表 |

**交互说明**：
- 点击"查看完整名单"进入人才名单页
- 人才名单支持按人才类别筛选

---

#### 模块：获奖情况（详细）

**功能描述**：展示学院人员获奖信息。

**展示内容**：

| 指标 | 说明 |
|------|------|
| 获奖总人次 | 近三年获奖人次 |
| 按年度分布 | 各年度获奖数量（柱状图） |
| 按级别分布 | 国家级/省部级/校级（环形图） |
| 获奖名单 | 完整获奖列表 |

**交互说明**：
- 支持按年度筛选
- 支持按获奖级别筛选
- 点击可查看获奖详情

---

### 3.3 全校对比功能

**功能描述**：展示本学院各指标在全校的相对位置。

**展示形式**：

| 形式 | 说明 |
|------|------|
| 排名徽章 | "第3/28"显示排名 |
| 区间分布 | "高于75%的学院"显示相对位置 |

**数据权限说明**：
- 院长可查看本学院所有数据明细
- 全校对比仅展示排名和区间，不展示其他学院具体数值
- 不支持查看其他学院的人员名单
