## 一、产品概述

### 1.1 产品背景

北京大学二级学院院长需要及时掌握本学院人事人才全貌，包括教职工结构、人才分布、人员流动等关键数据，以支撑日常管理和战略决策。

### 1.2 产品定位

面向二级学院院长的**移动端人事数据看板**，提供实时、直观、可穿透的人事数据分析，帮助院长快速掌握学院人才全貌，支持管理决策。

### 1.3 核心价值

| 价值点 | 说明 |
|--------|------|
| 随时随地 | 手机端随时查看，不依赖PC |
| 一目了然 | 核心KPI一眼掌握 |
| 数据穿透 | 从概览到明细逐层深入 |
| 横向对比 | 了解本学院在全校的相对位置 |

### 1.4 技术方案

- **形态**：移动端H5页面
- **适配**：iOS/Android手机浏览器
- **访问方式**：通过北大企业微信或链接访问
- **数据更新**：每日自动同步人事数据
- **开发工具**：基于BI工具开发，专注数据分析

### 3.2 引才质量指数

**计算公式**：
```
引才质量指数 = 博士占比×0.3 + 海外学缘占比×0.3 + 高层次人才占比×0.4
```

**展示内容**：
- 综合得分（0-100分）
- 全校排名
- 三个细分指标：博士占比、海外学缘占比、高层次人才占比
- **指标说明**：点击 ⓘ 图标显示计算公式和数据说明
