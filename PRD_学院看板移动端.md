## 一、产品概述

### 1.1 产品背景

北京大学二级学院院长需要及时掌握本学院人事人才全貌，包括教职工结构、人才分布、人员流动等关键数据，以支撑日常管理和战略决策。

### 1.2 产品定位

面向二级学院院长的**移动端人事数据看板**，提供实时、直观、可穿透的人事数据分析，帮助院长快速掌握学院人才全貌，支持管理决策。

### 1.3 核心价值

| 价值点 | 说明 |
|--------|------|
| 随时随地 | 手机端随时查看，不依赖PC |
| 一目了然 | 核心KPI一眼掌握 |
| 数据穿透 | 从概览到明细逐层深入 |
| 横向对比 | 了解本学院在全校的相对位置 |
| 智能洞察 | 数据驱动的决策建议 |

### 1.4 技术方案

- **形态**：移动端H5页面
- **适配**：iOS/Android手机浏览器
- **访问方式**：通过北大企业微信或链接访问
- **数据更新**：每日自动同步人事数据
- **开发工具**：基于BI工具开发

## 二、用户画像与使用场景

### 2.1 用户画像

| 角色 | 特征 | 核心需求 |
|------|------|---------|
| 二级学院院长 | 45-60岁，管理学院日常事务 | 快速了解学院人事全貌，支撑决策 |
| 学院书记 | 同上 | 关注党建、人才引进 |
| 分管副院长 | 协助院长管理 | 关注分管领域数据 |

### 2.2 核心使用场景

| 场景 | 频率 | 用户目标 | 设计重点 |
|------|------|---------|---------|
| 晨间速览 | 每日 | 快速了解学院最新动态 | 首页KPI一目了然 |
| 上级汇报 | 周/月 | 向校领导汇报学院情况 | 数据可截图、可导出 |
| 人才引进决策 | 不定期 | 评估现有人才结构缺口 | 人才分布、学缘分析 |
| 预算编制规划 | 年度 | 评估人员编制需求 | 编制使用率、拟退预警 |
| 学科建设规划 | 学期 | 评估学科发展情况 | 学科对比、人才分布 |

## 三、功能需求

### 3.1 功能架构（四Tab）

```
学院看板移动端（四Tab架构）
├── Tab1: 总览
│   ├── 核心KPI（在职、人才、获奖、编制使用率）
│   ├── 引才质量指数
│   ├── 人员结构环形图
│   └── 退休潮预警摘要
├── Tab2: 分析
│   ├── 年龄/性别/学历/职称/学缘/政治分布
│   ├── 人才分布
│   ├── 获奖分析
│   ├── 编制使用率详情
│   ├── 职称晋升漏斗
│   └── 学科发展对比
├── Tab3: 洞察
│   ├── 人才画像标签云
│   ├── 学术谱系分析
│   ├── 人员流动健康度
│   └── 人才投入产出
└── Tab4: 名单
    ├── 人员/人才/获奖筛选
    └── 可搜索列表
```

---

### 3.2 Tab1: 总览页面

**设计原则**：高信息密度、一屏可见、精致感

#### 3.2.1 核心KPI卡片

| 指标 | 展示内容 | 说明 |
|------|---------|------|
| 在职人数 | 456人 | 含同比变化 ↑2.3% |
| 专任教师 | 234人 | 含同比变化 ↑1.8% |
| 高层次人才 | 45人 | 含同比变化 ↑3人 |
| 本年获奖 | 28人次 | 含同比变化 ↑5人次 |
| 编制使用率 | 91.2% | 含预警状态 |
| 引才质量分 | 82分 | 全校排名 |

#### 3.2.2 引才质量指数

**计算公式**：
```
引才质量分 = 博士占比×0.3 + 海外学缘占比×0.3 + 高层次人才占比×0.4
```

**展示内容**：
- 综合得分（0-100分）
- 全校排名
- 三个维度的细分指标

#### 3.2.3 人员结构概览

**展示形式**：环形图
- 专任教师占比
- 职员占比
- 合同制占比
- 博士后占比
- 离退占比

#### 3.2.4 退休潮预警摘要

**展示内容**：
- 未来3年退休预测总数
- 正高退休人数（重点关注）
- 下一年退休高峰预警

---

### 3.3 Tab2: 分析页面

**设计原则**：图表为主、交互丰富、支持钻取

#### 3.3.1 八大分布维度

| 维度 | 图表类型 | 核心指标 |
|------|---------|---------|
| 年龄分布 | 柱状图 | 平均年龄、各年龄段人数 |
| 性别分布 | 环形图 | 男女比例 |
| 学历结构 | 环形图 | 博士/硕士/本科占比 |
| 职称结构 | 柱状图 | 正高/副高/中级/初级 |
| 学缘结构 | 树图 | 本校/国内/海外 |
| 政治面貌 | 环形图 | 党员/民主党派/群众 |
| 人才分布 | 柱状图 | 院士/长江/杰青/青年人才 |
| 获奖分析 | 柱状图+表格 | 按级别/年度分布 |

#### 3.3.2 编制使用率详情

**展示内容**：
- 总编制数、已使用、空余
- 按类别细分（专任教师、职员、合同制、博士后）
- 各类别使用率和预警状态

#### 3.3.3 职称晋升漏斗

**展示内容**：
- 入职→讲师→副教授→教授的晋升路径
- 各阶段平均晋升年限
- 与全校平均对比

#### 3.3.4 学科发展对比

**展示内容**：
- 各学科专任教师数量
- 各学科博士占比
- 各学科高层次人才数
- 学科短板预警

---

### 3.4 Tab3: 洞察页面

**设计原则**：深度分析、智能洞察、决策建议

#### 3.4.1 人才画像标签云

**展示内容**：
- 学历分布标签（博士68%、硕士22%...）
- 学缘分布标签（海外32%、本校19.5%...）
- 学科分布标签（经济学45%、金融学35%...）
- 人才分布标签（长江8人、杰青12人...）
- 特征分布标签（党员58%、45岁以下55%...）

#### 3.4.2 学术谱系分析

**展示形式**：桑基图或树图

**展示内容**：
- 教师毕业院校分布
- 学缘流向（从哪些院校到北大）
- 近亲繁殖预警

#### 3.4.3 人员流动健康度

**计算指标**：
- **流失率** = 年离职人数 / 平均在职人数
- **稳定性指数** = 平均在职年限
- **新鲜度** = 近3年入职人数占比

**展示内容**：
- 三个指标的得分和评级
- 近3年流动趋势图
- 流失风险预警

#### 3.4.4 人才投入产出

**展示内容**：
- 各类人才人数
- 各类人才获奖人次
- 人均获奖产出
- 优化建议

---

### 3.5 Tab4: 名单页面

**设计原则**：简洁、专注数据、可搜索

**筛选类型**：
- 全部
- 人员
- 人才
- 获奖

**人员卡片展示字段**：
- 姓名
- 职称/职位
- 最高学历
- 所属部门
- 来校时间
- 人才标签（如适用）

---

### 3.6 退休潮预警（详细）

**数据来源**：
- 出生日期（计算年龄）
- 离退时间（如有）
- 聘任专业技术职务（聘）（职称）
- 二级学科（学科分布）

**展示内容**：
- 未来3年退休预测总数
- 按年度分列
- 按职称分列（重点关注正高）
- 按学科分列
- 具体人员名单（可穿透）

---

### 3.7 编制使用率（详细）

**数据来源**：
- 人员编制（事业编制、企业编制、博士后）
- 人员类别

**展示内容**：
- 总编制数
- 已使用编制数
- 空余编制数
- 使用率百分比
- 按类别细分
- 预警状态（紧张/正常/充裕）

## 四、视觉规范

### 4.1 设计风格

采用 **Premium + Clean** 融合风格：
- Apple风格的精致感
- 简洁清晰的视觉层次
- 精确的间距系统（4/8/12/16/24/32）
- SVG线条图标（非emoji）

### 4.2 品牌色彩

| 用途 | 色值 | 说明 |
|------|------|------|
| 品牌主色 | #8C1515 | 北大红 |
| 成功色 | #16A34A | 绿色，正向变化 |
| 警告色 | #D97706 | 橙色，需要关注 |
| 危险色 | #DC2626 | 红色，负向变化 |
| 背景色 | #F8F9FA | 浅灰 |
| 卡片色 | #FFFFFF | 白色 |

### 4.3 图标规范

所有图标使用SVG线条图标（Lucide风格），不使用emoji。

## 五、数据需求

### 5.1 数据来源

| 数据 | 来源表 | 用途 |
|------|--------|------|
| 人员基本信息 | 人员基本信息宽表 | 人员统计、分布分析 |
| 编制信息 | 编制信息表 | 编制使用率 |

| 获奖信息 | 独立获奖表 | 获奖统计 |

| 角色 | 权限范围 |
|------|---------|
| 二级学院院长 | 本学院全部数据 + 全校匿名对比 |
| 学校管理员 | 全校数据 |
