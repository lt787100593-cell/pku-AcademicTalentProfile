# 北京大学学院看板 - 配色方案说明

## 设计原则

以**北大红（#8C1515）**为核心主题色，搭配中性色系，营造沉稳、专业的学术机构视觉风格。

---

## 核心色板

### 主色系（图表数据色）

| 序号 | 色值 | 色名 | 用途 |
|------|------|------|------|
| 1 | `#8C1515` | 北大红 | 主要数据、第一组数据 |
| 2 | `#B8860B` | 暗金色 | 第二组数据、对比数据 |
| 3 | `#2C3E50` | 深青灰 | 第三组数据 |
| 4 | `#708090` | 石板灰 | 第四组数据、辅助数据 |
| 5 | `#A0522D` | 赭石色 | 第五组数据 |

### 功能色

| 用途 | 色值 | 说明 |
|------|------|------|
| 正向/增长 | `#8C1515` | 使用主色表示正向指标 |
| 负向/减少 | `#708090` | 使用灰色表示负向指标 |
| 警告/注意 | `#B8860B` | 使用暗金色表示警告 |
| 成功/完成 | `#8C1515` | 使用主色表示成功状态 |

### 透明度变体

| 变体 | CSS写法 | 用途 |
|------|---------|------|
| 100%不透明 | `#8C1515` | 柱状图、实心元素 |
| 80%不透明 | `rgba(140, 21, 21, 0.8)` | 边框、强调 |
| 50%不透明 | `rgba(140, 21, 21, 0.5)` | 半透明背景 |
| 30%不透明 | `rgba(140, 21, 21, 0.3)` | hover效果 |
| 10%不透明 | `rgba(140, 21, 21, 0.1)` | 面积图填充、浅背景 |

## 图表应用场景

### 1. 环形图/饼图

**3分类数据**（如学历分布：博士/硕士/本科）
```javascript
backgroundColor: ['#8C1515', '#B8860B', '#708090']
```

**4分类数据**（如职称分布：正高/副高/中级/初级）
```javascript
backgroundColor: ['#8C1515', '#B8860B', '#2C3E50', '#708090']
```

### 2. 折线图

**单线趋势图**
```javascript
borderColor: '#8C1515',
pointBackgroundColor: '#8C1515',
backgroundColor: 'rgba(140, 21, 21, 0.1)'  // 面积填充
```

**双线对比图**
```javascript
// 线1（主）
borderColor: '#8C1515',
pointBackgroundColor: '#8C1515'

// 线2（辅）
borderColor: '#B8860B',
pointBackgroundColor: '#B8860B'
```

### 3. 柱状图

**分组柱状图**
```javascript
backgroundColor: ['#8C1515', '#B8860B', '#2C3E50']
```

**堆叠柱状图**
```javascript
backgroundColor: ['#8C1515', '#B8860B', '#708090']
```

### 4. 雷达图

```javascript
borderColor: '#8C1515',
backgroundColor: 'rgba(140, 21, 21, 0.2)',
pointBackgroundColor: '#8C1515'
```

## 色彩搭配规则

### 主要规则

1. **主色优先**：最重要的数据使用北大红 `#8C1515`
2. **对比明确**：双线对比使用 `#8C1515` + `#B8860B`，明暗对比清晰
3. **层次分明**：多分类数据按 1→5 的顺序依次使用色板
4. **功能统一**：正向用主色，负向用灰色

### 避免事项

- ❌ 避免使用高饱和度的纯色（如 #FF0000、#00FF00）
- ❌ 避免使用过多颜色（超过5种）
- ❌ 避免颜色过于接近（如 #8C1515 和 #DC2626）
- ❌ 避免在深色背景上使用深色数据

## 文件引用

CSS变量定义在 `styles.css` 的 `:root` 中：

```css
:root {
    --primary: #8C1515;
    --primary-light: #A52A2A;
    --gold: #B8860B;
    --dark-gray: #2C3E50;
    --slate: #708090;
    --sienna: #A0522D;
}
```
