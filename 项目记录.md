## 一、项目背景

### 1.1 需求来源
为北京大学二级学院院长设计移动端人事数据看板，帮助院长快速掌握学院人才全貌，支撑管理决策。

### 1.2 用户画像
- 二级学院院长（45-60岁）
- 学院书记
- 分管副院长

### 1.3 使用场景
- 晨间速览：每日快速了解动态
- 上级汇报：周/月汇报数据
- 人才引进：评估结构缺口
- 预算编制：评估编制需求

## 三、功能架构（最终版）

### 3.1 四Tab架构

```
学院看板移动端
├── Tab1: 总览
│   ├── 核心KPI卡片（在职、人才、获奖）
│   ├── 引才质量指数（含指标说明）
│   ├── 编制使用率（含详情表格）

│   ├── 趋势分析（在职人数双线图、博士占比、人才分组柱状图、获奖堆叠图）
│   └── 对比分析（全校雷达图、学科对比）
├── Tab3: 洞察
│   ├── 人才画像标签云
│   ├── 学缘来源分析（TOP5院校）
│   ├── 人员流动指标

    ├── 可搜索列表
    └── 获奖详情（级别、项目、时间）
```

## 四、核心功能说明

### 4.1 总览页面
| 模块 | 说明 |
|------|------|
| 核心KPI | 在职人数、专任教师、高层次人才、本年获奖，含同比环比变化 |
| 引才质量指数 | 综合评分 = 博士占比×0.3 + 海外学缘×0.3 + 高层次人才×0.4，点击ⓘ可查看说明 |
| 编制使用率 | 进度条 + 按类别详情表格（专任教师/职员/合同制/博士后） |
| 退休潮预测 | 未来3年退休预测，分正高/副高显示人数 |
| 人才概况 | 院士、长江学者、杰青、青年人才人数 |

### 4.2 分析页面
| 模块 | 说明 |
|------|------|
| 分布结构分析 | 8大维度Tab切换，每个维度有图表+表格 |
| 趋势分析 | 近5年在职人数（双线图）、博士占比、高层次人才（分组柱状图）、获奖（堆叠图） |
| 对比分析 | 全校对比雷达图（6维度）、学科发展对比表格（含晋升年限） |

### 4.3 洞察页面
| 模块 | 说明 |
|------|------|
| 人才画像标签云 | 可视化学历、学缘、学科、人才、特征分布 |
| 学缘来源分析 | TOP5毕业院校 + 本校学缘与全校对比 |
| 人员流动指标 | 近三年流失率、平均在职年限、新进人员占比 |
| 人才获奖产出 | 各类人才人数、获奖人次、人均获奖 |

### 4.4 名单页面
| 模块 | 说明 |
|------|------|
| 多维度筛选 | 全部人员、人员、人才、获奖 |
| 搜索功能 | 按姓名、职称、学历、部门搜索 |
| 获奖详情 | 级别（国家级/省部级/校级）、名称、项目、时间 |
| 人事信息 | 姓名、职称、学历、部门、来校时间、人才类别 |

## 八、设计决策记录

### 8.1 数据展示
- 不做AI解读，专注数据可视化（因为基于BI工具开发）
- 所有指标使用完整清晰的名称（如"近三年人员流失率"）
- 排名显示为"第X/多学院"，不编造具体学院数量

### 8.2 交互设计
- 切换Tab后页面滚动到顶部
- 所有可点击卡片有按压反馈（scale 0.98）
- 图表支持点击穿透到人员名单

### 8.3 视觉设计
- 使用SVG图标，不使用emoji
- 北大红作为品牌主色
- Premium风格：毛玻璃、渐变边框、悬停动效

## 十三、V2版本（双单位类型）

### 13.1 版本说明

V2版本基于用户反馈，新增**职能部门**页面，同时优化**二级学院**页面：

| 变更项 | V1 | V2 |
|--------|----|----|
| 编制使用率 | 有 | **移除**（对学院无意义） |
| 全校对比雷达图 | 有 | **移除**（人才数据不准） |
| 学科发展对比 | 有 | **保留**（宏观数据可计算） |
| 各维度全校排名 | 有 | **移除** |
| 单位类型 | 仅学院 | **新增**：学院 + 职能部门 |

### 13.2 二级学院页面

**适用单位**：光华管理学院、经济学院、法学院、国际关系学院、信息管理系等

**Tab1：总览**
- 核心KPI（在职人数、专任教师、高层次人才、本年获奖）含同比环比
- 引才质量指数（含指标说明）
- 退休潮预测
- 人才概况（院士/长江/杰青/青年人才）
- 获奖亮点

**Tab2：分析**
- 分布结构分析（8维度Tab：年龄/性别/学历/职称/学缘/政治/人才/获奖）
- 趋势分析（近5年在职人数双线图、博士占比、高层次人才分组柱状图、获奖堆叠图）
- 学科发展对比（含晋升年限）

**Tab3：洞察**
- 人才画像标签云
- 学缘来源分析（TOP5院校）
- 人员流动指标
- 人才获奖产出分析

**Tab4：名单**
- 人员/人才/获奖筛选
- 可搜索列表 + 获奖详情

### 13.3 职能部门页面（新增）

**适用单位**：计算中心、人事部、教务部、科研部、财务处等

**核心差异**：不承担教学任务，无"专任教师"和"高层次人才"概念

**Tab1：总览**
- 核心KPI（在职人数、本年获奖）含同比环比
- 人员构成（职员/合同制/博士后 占比）
- 退休潮预测
- 获奖亮点

**Tab2：分析**
- 分布结构分析（6维度Tab：年龄/性别/学历/职称/政治面貌/用工类型）
- 趋势分析（近5年在职人数变化、用工类型变化）
- 学科发展对比（含晋升年限）

**Tab3：洞察**
- 人员流动指标（流失率、平均在职年限、新进占比）
- 获奖情况汇总（按级别统计）
- 职级结构（正高/副高/中级/其他 四级金字塔图）

**Tab4：名单**
- 人员/获奖筛选
- 可搜索列表 + 获奖详情
